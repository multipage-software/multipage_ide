# multipage_ide
Multipage IDE with Maclan macrolanguage. Development of software elements.

You are welcome to make clones of the source code so that the IDE can be improved together and developed. It could be bridge to Python, better intellisense, perfomance and GUI or other improvements that I cannot imagine yet. Please send me an e-mail to root@multipage-software.org so that I can add you to the list of contributors on GitHub.
